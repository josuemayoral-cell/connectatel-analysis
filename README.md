# 📊 Análisis de Comportamiento de Usuarios - ConnectaTel

## 🎯 Descripción del Proyecto
Análisis exploratorio de datos de una empresa de telecomunicaciones latinoamericana para identificar patrones de uso, segmentar clientes y optimizar la oferta comercial basada en comportamiento real de usuarios.

## 📈 Objetivos
- Segmentar usuarios por comportamiento de uso y demografía
- Identificar oportunidades de optimización en planes actuales
- Detectar patrones de uso extremo y comportamientos atípicos
- Generar recomendaciones comerciales basadas en datos

## 📊 Datasets Utilizados
- **plans.csv**: Información de planes actuales (precios, minutos, GB incluidos)
- **users_latam.csv**: Datos de clientes (edad, ciudad, plan, fechas)
- **usage.csv**: Registro de uso real (llamadas y mensajes del 2024)

## 🔍 Principales Hallazgos
- **Paradoja comercial**: Usuarios del plan Básico usan MÁS servicios que Premium
- **Segmentación por edad**: Jóvenes usan más minutos de llamada de lo esperado
- **Outliers identificados**: Usuarios con >60 minutos, >11 mensajes, >10 llamadas
- **Oportunidad de mercado**: 64.88% en plan Básico vs 35.12% en Premium
